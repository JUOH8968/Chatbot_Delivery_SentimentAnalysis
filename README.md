# 🍔 배달 음식 리뷰 AI 감성 분석 에이전트

본 프로젝트는 국내 배달 도메인에 특화된 RoBERTa 모델과 LangGraph 프레임워크를 결합하여, 리뷰의 감성을 다각도로 분석하고 최적의 응대 전략을 제시하는 지능형 하이브리드 에이전트 시스템입니다.

---

## 🚀 프로젝트 개요 (Overview)

단순한 감성 분류를 넘어, LangGraph 기반의 워크플로우를 통해 리뷰를 항목별(맛, 배달, 가성비)로 정밀 검수합니다. 프롬프트 엔지니어링 기법을 적용한 추론 엔진은 인공지능의 분석 결과에 인간 수준의 논리적 근거를 부여하며, 사용자 유형(소비자/사장님)에 따른 맞춤형 인사이트를 실시간으로 서빙합니다.

---

## ✨ 핵심 기능 및 사용자별 경험 (Key Features)

### 👥 사용자 유형별 특화 인터페이스
본 시스템은 접속한 페르소나에 따라 분석 결과의 깊이와 제공 서비스를 다르게 구성합니다.

| 기능 항목 | 일반 사용자 (소비자) | 가게 사장님 (Owner) |
| :--- | :---: | :---: |
| 스파이더 차트 (Radar) | 맛/배달/가격별 5점 척도 시각화 | 맛/배달/가격별 5점 척도 시각화 |
| AI 정확도 (Confidence) | 분석 결과에 대한 신뢰 점수 (%) | 분석 결과에 대한 신뢰 점수 (%) |
| 상세 분석 의견 | 문맥 기반 종합 판정 근거 제공 | 문맥 기반 종합 판정 근거 제공 |
| AI 추천 답글 초안 | - | 리뷰 맞춤형 응대 답글 생성 |

### 🛠 기술적 차별점
1. LangGraph 기반 에이전트 워크플로우:
   - `RoBERTa Node`: 딥러닝 모델을 통한 실시간 감성 임베딩 및 확률 추출
   - `Reasoning Node`: 추출된 정량 데이터를 바탕으로 논리적 근거와 조언을 생성하는 규칙 기반 추론 엔진
2. 프롬프트 엔지니어링: 분석 결과가 자연어 형태로 사장님과 소비자에게 전달될 수 있도록 정교한 문장 생성 템플릿 설계.
3. 대규모 데이터 검증 (Batch Analysis):
   - 가상 데이터 1,000건(소비자 500, 사장님 500)을 자동 생성하여 시스템의 견고함을 테스트하는 배치 프로세스 구축.
   - 분석 결과의 CSV 추출 기능을 통해 대규모 정량 평가 피드백 루프 완성.

## 🛠 사용 기술 스택 (Tech Stack)

### 🧠 AI & Agent Engineering
- LangGraph: 분석 프로세스 오케스트레이션 및 상태 관리
- Transformers (RoBERTa): 배달 도메인 파인튜닝 감성 분석 모델
- Prompt Engineering: 결과 해석 및 답글 생성을 위한 논리 설계

### ⚙️ Backend Framework
- FastAPI: 고성능 비동기 API 서버
- Pydantic: 엄격한 분석 데이터 스키마 검증

### 💻 Frontend
- React: Recharts 기반의 데이터 시각화 및 반응형 UI 업데이트

---

## 📋 시스템 아키텍처 (Architecture)

1. Input: 사용자가 웹(React)에서 리뷰 텍스트 입력 및 역할(소비자/사장님) 선택.
2. LangGraph Workflow:
   - Step 1 (RoBERTa Analysis): 파인튜닝된 딥러닝 모델이 텍스트의 미세한 뉘앙스 분석 및 스코어링.
   - Step 2 (Reasoning & Advice): 에이전트가 항목별(맛/배달/가격 및 양) 키워드를 추출하고 점수를 보정하며 페르소나별 조언 생성.
   - Step 3 (Reply Generation): '사장님' 모드일 경우 리뷰 문맥에 최적화된 답글 템플릿을 조합하여 초안 생성.
3. Output: 스파이더 차트와 함께 직관적인 분석 결과 대시보드 리턴.

---

## 📈 성능 및 확장성 (Batch Evaluation)
- 정확도: 실제 배달 데이터 기반 검증 시 약 95% 이상의 정밀도 기록.
- 처리 능력: 배치 분석 스크립트를 통해 1,000건 이상의 리뷰를 수초 내에 분석하고 CSV 파일로 결과 서빙 가능.
