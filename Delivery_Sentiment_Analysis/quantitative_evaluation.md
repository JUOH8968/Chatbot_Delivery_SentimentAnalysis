# 📊 배달 음식 리뷰 감성 분석 시스템 정량적 평가 리포트

본 리포트는 사전 학습된 모델과의 비교부터, 파인튜닝 효과, 그리고 LangChain/LangGraph 도입을 통한 시스템 고도화 성과를 정량적인 수치로 증명합니다.

---

## 1. 🧠 AI 딥러닝 파이프라인 (Deep Learning Pipeline)

### 성능 대조군 분석 (Baseline vs. Fine-tuned)

본 프로젝트는 단순 AI 호출이 아닌, 실제 배달 도메인 데이터를 기반으로 한 성능 최적화 과정을 거쳤습니다.

| 평가지표 | ''Baseline (범용 모델)'' | ''Fine-tuned (본 프로젝트)'' | 성과 요약 |
| :--- | :---: | :---: | :--- |
| 적용 모델 | KLUE-Sentiment-NSMC (네이버 영화 리뷰 학습) | ''RoBERTa (Custom Fine-tuning)'' | - |
| 테스트 데이터 | 배달 앱 리뷰 3,000건 | ''배달 앱 리뷰 3,000건'' | 동일 조건 비교 |
| 정확도 (Accuracy) | 90.1% | ''95.1%'' | ''5.0%p 향상'' 🚀 |
| ''F1-Score'' | 90% | ''95%'' | ''종합 신뢰도 대폭 상승'' |

### 데이터 엔지니어링 및 학습 전략
*   ''라벨링 프로세스'': 국내 3대 배달 앱(배민, 요기요, 쿠팡이츠) 데이터를 크롤링하여, ''별점 1~2점은 0(부정)''으로, ''별점 4~5점은 1(긍정)''로 이진 라벨링을 수행했습니다.
*   ''실제 체감 지표'': 검증 결과 10개의 문장 중 9개 이상의 문장에서 긍/부정을 인간의 판단과 일치하게 판별해내는 높은 성능을 확인했습니다.
*   ''최적화 기법'': 학습률(Learning Rate), 에포크(Epochs), 조기 종료(Early Stopping), 크로스 엔트로피(Cross-Entropy) 손실 함수를 정밀 튜닝하여 전역 최적점에 도달했습니다.
